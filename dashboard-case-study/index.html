Case Study — Portfolio Analytics

Portfolio Operations Dashboard

Built at Wintrust Financial Corporation to manage a $200M premium finance portfolio across 75–100 concurrent agency relationships · Jason Santos

The Problem

The portfolio had no formal CS infrastructure. Engagement health, onboarding progress, and renewal risk across 75–100 agencies lived in scattered exports and manual review. There was no single view that let me prioritize which accounts needed attention this week, versus which were healthy and could wait.

What I Built

A tracking system that started in Excel — pivot tables and manual scoring — and was rebuilt in Power BI as the portfolio grew past the point Excel could handle cleanly. The dashboard consolidated four things I was previously tracking separately: portfolio-level trends, agency-level activity (quote volume and transaction frequency, the leading indicators of disengagement), onboarding progress against the tiered framework, and renewal pipeline risk. The goal was a single view I could open Monday morning and know exactly which 5–10 accounts needed a call that week.

Evolution: Manual to Automated

Before — Excel

How the tracker started

A running workbook: one tab per agency-tracking dimension, manual pivot tables rebuilt weekly, scoring done by eye against quote volume and transaction counts. Functional up to roughly 40-50 agencies, then update time started competing with actual account work.

After — Power BI

Where it landed

Same underlying logic — task sequencing and account prioritization by activity signal — rebuilt as a connected model that refreshed automatically instead of by hand. What used to be a Sunday-night rebuild became a five-minute Monday check.

How It Was Used

Renewal conversations started from the pipeline view, not from memory — accounts flagged "High" or "Watch" got outreach before the renewal date, not after activity had already dropped for a quarter. The onboarding view replaced a running spreadsheet I'd update by hand; anyone could see which agencies were behind pace without asking me. Portfolio trend and activity views were what I opened first each Monday to decide where the week's attention went.

A note on the data: the dashboards above use illustrative sample data — agency names, exact figures, and renewal dates are fabricated for this walkthrough, since the underlying Wintrust data is proprietary. The structure, metrics, and workflow shown reflect the actual dashboards I built and used to manage the portfolio, including the shift from a manual Excel-based tracker to Power BI as the process scaled.
